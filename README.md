## Практическая работа №9 "Docker Compose"

Данный репозиторий представляет собой docker-compose конфигурацию, которая включает три Docker-образа: два Python-приложения (сервера) и балансировщик нагрузки на основе nginx.

### Назначение системы:

Балансировщик nginx выполняет терминацию SSL-соединений

Перенаправляет зашифрованный трафик на внутренние веб-серверы

### Цель работы:

После перехода по адресу https://localhost и подтверждения самоподписанного сертификата, вы должны увидеть [страницу с приветствием](output.png). При многократном обновлении страницы контент должен поочередно возвращаться то от app1, то от app2, демонстрируя работу балансировщика нагрузки.

### Задача:

В текущей конфигурации содержит большое количество ошибок (разного уровня сложности). Ваша задача - провести детальный анализ, выявить и исправить все проблемы в системе. Найденные проблемы и их исправления отразить в итоговом отчете и комментариях в коде конфигурации.
